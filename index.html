Page Flip Book

A clean and customizable page flip effect for your website.

Customization Instructions:

  • Add your photos: Place your page images in the 'images' folder (e.g., page1.jpg, page2.jpg, etc.) and update the JavaScript array with the correct filenames.
  • Images should be around 400x600 pixels for best results, but the CSS will scale them.
  • Adjust the CSS in page-flip.css for styling (colors, sizes, etc.).
  • Modify the HTML structure above for layout changes.
  • The audio file 'page-flip.mp3' can be replaced with your own sound effect.
